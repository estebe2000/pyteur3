# 🐍 Pyteur OS – Documentation Utilisateur

![Pyteur OS Illustration](https://upload.wikimedia.org/wikipedia/commons/thumb/0/0a/Python.svg/1200px-Python.svg.png)

Bienvenue sur la documentation officielle de **Pyteur OS**, la plateforme éducative open-source pour l’enseignement de l’informatique au lycée (NSI/SNT).

> **Pyteur OS** centralise tous les outils pédagogiques : gestion de classes, création d’exercices, QCM, IA, drive, messagerie, statistiques, bac à sable Python, et bien plus.
